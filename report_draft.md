# GIS와 Python을 활용한 지반 안정성 평가 및 도시계획 의사결정 모델 보고서 초안

## 1. 주제 설정 및 연결 구조화

### 우리가 달성하고자 하는 내용

본 프로젝트의 목표는 실제 공간 데이터를 활용하여 지역별 지반 안정성 위험도를 계산하고, 그 결과를 도시계획 의사결정에 활용할 수 있는 탐구용 알고리즘을 구현하는 것이다. 지반 안정성은 건축물의 안전, 주거지 개발, 공공시설 입지 선정, 재난 예방과 밀접하게 관련된다. 따라서 지반 또는 토질 자료와 지하수 자료를 GIS 공간 분석 기법으로 격자 단위에 연결하고, Python 알고리즘을 이용해 위험도 점수를 계산하였다.

현재 확보한 자료는 국토교통부 지반정보 기반의 시추 지층 자료이며, 이 자료에 포함된 `USCS_NM`은 토질 정보, `GWL_M`은 지하수위 또는 지하수면 깊이 정보로 활용하였다. 이후 단층, 지질, 경사도, 풍화 정도 자료를 추가하면 같은 구조 안에서 더 정교한 지반 안정성 평가 모델로 확장할 수 있도록 설계하였다.

### 연결 유형

지역사회문제-환경-교과

### 연결된 핵심 요소

A: 도시계획 및 토지 이용  
B: 지반 안정성, 토질, 지하수위  
C: Python, GIS, 공간 데이터 분석

### 연결 구조 설명

도시계획에서 건물이나 공공시설을 어디에 배치할지 결정하려면 지반의 안정성을 고려해야 한다. 지반이 약하거나 지하수위가 높은 지역은 침하, 균열, 사면 불안정 등의 위험이 커질 수 있다. 본 탐구는 시추 지층 자료와 지하수위 자료를 공간 좌표와 연결하고, 이를 격자 단위로 집계하여 위험도 점수를 계산한다. 즉 환경적 요소인 지반 조건을 컴퓨터공학의 데이터 처리와 알고리즘으로 분석하여 도시계획 의사결정에 연결하는 구조이다.

## 2. 문제 정의

### 구체적 문제

시추 지층 자료와 지하수위 자료를 이용하여 지역별 지반 위험도를 정량적으로 계산하고, 위험도 등급에 따라 적절한 토지 이용 방안을 추천할 수 있는가?

### 이해 관계자 분석

도시계획 담당자는 안전한 개발 가능 지역을 찾기 위해 지반 정보를 필요로 한다. 건축 관계자는 건축물 기초 설계와 부지 선정 단계에서 지반 안정성 정보를 활용할 수 있다. 지역 주민은 지반 위험도가 낮은 안전한 주거환경을 원한다. 환경 및 방재 담당자는 위험도가 높은 지역을 녹지, 완충지대, 방재시설 등으로 관리할 필요가 있다. 따라서 지반 위험도 분석 결과는 행정기관, 건축 분야, 주민, 방재 분야 모두에게 의미가 있다.

## 5. 데이터 수집 및 기초 분석

### 주제

국토교통부 지반정보 기반 시추 지층 자료를 활용한 격자 단위 지반 안정성 위험도 분석

### 자료

수집한 자료는 `jiban_strata_WGS84.shp` 형식의 공간 데이터이다. 이 자료는 시추공 위치와 지층 정보를 포함하며, 약 987,632개의 지층 레코드가 포함되어 있다. 주요 속성은 다음과 같다.

| 컬럼명 | 의미 | 본 탐구에서의 활용 |
|---|---|---|
| `BH_CODE` | 시추공 코드 | 위치 및 지층 구분 |
| `LAYER_CD` | 지층 코드 | 지층 정보 참고 |
| `START_M` | 지층 시작 깊이 | 지층 깊이 정보 |
| `END_M` | 지층 종료 깊이 | 지층 깊이 정보 |
| `THICK_M` | 지층 두께 | 지층 특성 참고 |
| `CIVIL_NM` | 토목 지층명 | 토질 특성 참고 |
| `USCS_NM` | 통일분류법 기반 토질명 | 토질 위험도 점수 계산 |
| `GWL_M` | 지하수위 또는 지하수면 깊이 | 지하수 위험도 점수 계산 |
| `SRC_X`, `SRC_Y` | 원본 좌표 | 위치 정보 확인 |

### 자료의 의미

`USCS_NM`은 흙의 종류를 나타내므로 지반이 상대적으로 안정적인지, 연약하거나 침하 가능성이 있는지를 판단하는 데 사용할 수 있다. 예를 들어 자갈질이나 암반에 가까운 토양은 낮은 위험 점수를 부여하고, 점토질 또는 유기질 토양은 높은 위험 점수를 부여한다. `GWL_M`은 지하수면의 깊이를 나타내며, 지하수면이 얕을수록 지반 안정성에 불리할 수 있다고 가정한다. 따라서 이 자료는 지반 안정성 평가의 핵심 자료로 활용될 수 있다.

## 6. 교과 심화 탐색: 모델링

### 주제

GIS 공간 분석과 Python 알고리즘을 이용한 격자 기반 지반 위험도 모델링

### 모델링 과정

먼저 시추 지층 자료를 불러오고, `region.shp`가 없는 상황에서도 분석이 가능하도록 데이터 분포 범위를 기준으로 자동 격자를 생성하였다. 생성된 격자는 약 7.4km 크기의 정사각형 분석 단위이며, 총 1,720개 격자가 분석에 사용되었다.

각 격자 안에 포함된 시추 지층점을 공간 결합하였다. 토질 자료는 격자 안에서 가장 많이 나타나는 `USCS_NM` 값을 대표 토질로 선택하였다. 지하수위 자료는 격자 안에 포함된 `GWL_M` 값의 평균을 계산하였다. 이후 토질 위험 점수와 지하수 위험 점수를 각각 0~100점으로 변환하였다.

원래 전체 모델의 가중치는 단층 0.25, 지질 0.20, 토질 0.20, 지하수 0.15, 경사도 0.10, 풍화 정도 0.10으로 설계하였다. 그러나 현재는 토질과 지하수 자료만 확보되어 있으므로 사용 가능한 요소의 가중치만 합산하여 다시 100%가 되도록 재정규화하였다. 이를 통해 자료가 일부만 있는 상황에서도 위험도 점수가 지나치게 낮게 계산되는 문제를 줄였다.

현재 실행에서 사용된 핵심 계산식은 다음과 같다.

```text
final_risk_score =
soil_score * 재정규화된 토질 가중치 +
groundwater_score * 재정규화된 지하수 가중치
```

전체 자료가 모두 준비되면 다음 계산식으로 확장된다.

```text
final_risk_score =
fault_score * 0.25 +
geology_score * 0.20 +
soil_score * 0.20 +
groundwater_score * 0.15 +
slope_score * 0.10 +
weathering_score * 0.10
```

위험도 등급은 기존 4단계보다 결과 차이를 더 자세히 보기 위해 7단계로 세분화하였다.

| 점수 범위 | 위험도 등급 |
|---|---|
| 0 이상 15 미만 | 매우 낮음 |
| 15 이상 30 미만 | 낮음 |
| 30 이상 45 미만 | 약간 낮음 |
| 45 이상 60 미만 | 보통 |
| 60 이상 75 미만 | 높음 |
| 75 이상 90 미만 | 매우 높음 |
| 90 이상 100 이하 | 극히 높음 |

### 모델링 의미

이 모델은 실제 지반 안정성을 확정적으로 판단하는 도구가 아니라, 공간 데이터를 기반으로 위험 가능성을 비교하는 탐구용 의사결정 모델이다. 한정된 자료를 격자 단위로 집계함으로써 지역별 비교가 가능해졌고, 토질과 지하수위라는 서로 다른 자료를 하나의 위험도 점수로 통합하였다. 또한 나중에 단층, 지질, 경사도, 풍화 정도 자료가 추가되면 같은 구조 안에서 더 정교한 모델로 확장할 수 있다.

## 7. 교과 심화 탐색: 최적화

### 주제

누락 자료가 있는 상황에서도 실행 가능한 지반 위험도 평가 알고리즘 최적화

### 최적화 내용

첫째, `region.shp`가 없을 경우 자동으로 격자를 생성하도록 알고리즘을 개선하였다. 이를 통해 행정구역 자료가 없어도 분석을 진행할 수 있었다.

둘째, 누락된 자료가 있을 때 프로그램이 중단되지 않고 경고 메시지를 출력한 뒤 계속 실행되도록 하였다. 실제 공공데이터 분석에서는 모든 자료가 한 번에 준비되지 않는 경우가 많기 때문에 이는 확장성과 안정성을 높이는 최적화이다.

셋째, 사용 가능한 요소만으로 가중치를 재정규화하였다. 기존 방식에서는 토질과 지하수만 사용할 경우 전체 점수가 낮은 범위에 몰릴 수 있다. 이를 해결하기 위해 현재 사용 가능한 요소의 가중치 합을 100%로 다시 조정하였다.

넷째, 위험도 등급을 4단계에서 7단계로 세분화하였다. 이를 통해 지역 간 차이를 더 자세히 확인할 수 있었다.

## 8. 탐구 결과

분석 결과 총 1,720개의 격자가 생성되었고, 각 격자별로 토질 위험도, 지하수 위험도, 최종 위험도 점수, 위험도 등급, 추천 토지 이용 방향이 계산되었다.

위험도 등급 분포는 다음과 같다.

| 위험도 등급 | 격자 수 |
|---|---:|
| 매우 낮음 | 20 |
| 낮음 | 134 |
| 약간 낮음 | 356 |
| 보통 | 591 |
| 높음 | 576 |
| 매우 높음 | 43 |
| 극히 높음 | 0 |

분석 결과 `보통`과 `높음` 등급의 격자가 가장 많이 나타났다. 이는 현재 사용한 자료에서 토질 조건과 지하수위 조건이 지역별로 차이를 보였기 때문이다. 특히 지하수위가 얕거나 상대적으로 불안정한 토질이 대표값으로 선택된 격자는 높은 위험도 점수를 나타냈다.

출력 결과는 `final_result.csv`, `final_result.geojson`, `risk_map.html`, `risk_grade_summary.png`로 저장되었다. `risk_map.html`은 Folium을 사용한 지도 파일로, 각 격자의 위험도 등급을 색상으로 확인할 수 있다. `final_result.csv`에서는 격자명, 토질 점수, 지하수 점수, 최종 위험도 점수, 위험도 등급, 추천 토지 이용 내용을 표로 확인할 수 있다.

## 9. 탐구 결론

본 탐구를 통해 GIS 공간 데이터와 Python 알고리즘을 활용하면 지반 안정성과 관련된 정보를 지역 단위로 정량화할 수 있음을 확인하였다. 특히 시추 지층 자료의 토질 정보와 지하수위 정보를 이용하여 격자별 위험도 점수를 계산하고, 위험도 등급에 따라 토지 이용 방향을 추천할 수 있었다.

탐구 결과는 실제 도시계획 의사결정에서 기초 참고 자료로 활용될 가능성을 보여준다. 예를 들어 위험도가 낮은 지역은 주거지나 공공시설 입지 후보로 검토할 수 있고, 위험도가 높은 지역은 저밀도 개발, 공원, 완충지대, 방재시설 등으로 관리하는 방향을 고려할 수 있다.

다만 본 결과는 탐구용 모델이며 실제 지반 안정성 판정으로 단정할 수 없다. 현재 분석에는 단층, 지질, 경사도, 풍화 정도 등의 자료가 포함되지 않았기 때문에 해석에 한계가 있다. 향후 해당 자료를 추가하고, 전문가 기준이나 문헌 자료를 바탕으로 점수표와 가중치를 보정한다면 더 신뢰도 높은 GIS 기반 도시계획 의사결정 모델로 발전시킬 수 있다.
